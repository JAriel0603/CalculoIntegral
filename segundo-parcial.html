Tema 01

Área bajo una curva

Concepto

El área bajo una curva representa la región comprendida entre la gráfica de una función, el eje horizontal y dos límites establecidos. Esta área se calcula mediante una integral definida.

Fórmula principal

A = ∫ₐᵇ f(x) dx

Ejercicio desarrollado

Calcular el área bajo la función: f(x) = 2x + 1, en el intervalo [0, 3].

A = ∫₀³ (2x + 1) dx

A = [x² + x]₀³

A = (3² + 3) − (0² + 0)

A = 9 + 3

Resultado: A = 12 unidades²

Reflexión

Este tema permitió comprender que una integral definida puede utilizarse para medir regiones geométricas. También fue importante identificar correctamente los límites de integración.

Tema 02

Área entre curvas

Concepto

El área entre dos curvas se obtiene restando la función inferior de la función superior y calculando la integral definida dentro del intervalo correspondiente.

Fórmula principal

A = ∫ₐᵇ [f(x) − g(x)] dx

Ejercicio desarrollado

Calcular el área entre: f(x) = 2x y g(x) = x², en [0, 2].

A = ∫₀² (2x − x²) dx

A = [x² − x³/3]₀²

A = 4 − 8/3

Resultado: A = 4/3 unidades²

Reflexión

Para resolver este tipo de problemas es necesario determinar cuál función se encuentra arriba y cuál se encuentra abajo dentro del intervalo estudiado.

Tema 03

Sólidos de revolución

Concepto

Un sólido de revolución se forma cuando una región plana gira alrededor de un eje. Su volumen puede determinarse mediante los métodos de discos, arandelas o capas cilíndricas.

Fórmula del método de discos

V = π ∫ₐᵇ [f(x)]² dx

Ejercicio desarrollado

Calcular el volumen generado por: f(x) = x, en [0, 2], alrededor del eje x.

V = π ∫₀² x² dx

V = π [x³/3]₀²

V = π(8/3)

Resultado: V = 8π/3 unidades³

Reflexión

Este tema permitió relacionar las integrales con objetos tridimensionales. La elección del método depende del eje de rotación y de la forma de la región.

Tema 04

Longitud de arco

Concepto

La longitud de arco permite calcular la distancia existente a lo largo de una curva entre dos puntos determinados.

Fórmula principal

L = ∫ₐᵇ √[1 + (f′(x))²] dx

Ejercicio desarrollado

Calcular la longitud de: f(x) = 3x, en [0, 2].

f′(x) = 3

L = ∫₀² √(1 + 3²) dx

L = ∫₀² √10 dx

L = [x√10]₀²

Resultado: L = 2√10 unidades

Reflexión

La longitud de una curva depende tanto de la función como de su derivada. Este procedimiento permite medir recorridos que no pueden calcularse únicamente con fórmulas de líneas rectas.

Tema 05

Aplicaciones en cinemática

Concepto

En cinemática, la integración permite obtener la velocidad a partir de la aceleración y la posición a partir de la velocidad.

Relaciones principales

v(t) = ∫ a(t) dt
s(t) = ∫ v(t) dt

Ejercicio desarrollado

Dada la velocidad: v(t) = 4t + 2

Determinar la función de posición, considerando s(0) = 3.

s(t) = ∫ (4t + 2) dt

s(t) = 2t² + 2t + C

s(0) = 3

C = 3

Resultado: s(t) = 2t² + 2t + 3

Reflexión

Esta aplicación demuestra que el cálculo integral permite analizar situaciones reales de movimiento. Las condiciones iniciales son necesarias para determinar la constante de integración.
