Tema 01

Integrales definidas e indefinidas

Concepto

La integral indefinida representa una familia de funciones cuya derivada coincide con el integrando. Por esta razón, su resultado incluye una constante de integración.

La integral definida permite calcular la acumulación de una función entre dos límites.

Fórmulas principales

∫ f(x) dx = F(x) + C
∫ₐᵇ f(x) dx = F(b) − F(a)

Ejercicio desarrollado

Resolver: ∫ (5x⁴ − 6x² + 3) dx

= ∫ 5x⁴ dx − ∫ 6x² dx + ∫ 3 dx

= x⁵ − 2x³ + 3x + C

Resultado: x⁵ − 2x³ + 3x + C

Reflexión

Este tema permitió comprender la relación inversa existente entre la derivación y la integración.

Tema 02

Integración por partes

Concepto

La integración por partes se aplica principalmente cuando el integrando contiene el producto de dos funciones.

Fórmula principal

∫ u dv = uv − ∫ v du

Ejercicio desarrollado

Resolver: ∫ x eˣ dx

u = x

du = dx

dv = eˣ dx

v = eˣ

∫ x eˣ dx = xeˣ − ∫ eˣ dx

Resultado: xeˣ − eˣ + C

Reflexión

La correcta selección de las variables u y dv permite simplificar el procedimiento.

Tema 03

Cambio de variable

Concepto

El cambio de variable transforma una integral compleja en una expresión más sencilla mediante una nueva variable.

Fórmula principal

u = g(x)    y    du = g′(x) dx

Ejercicio desarrollado

Resolver: ∫ 3x²(x³ + 4)⁵ dx

u = x³ + 4

du = 3x² dx

∫ u⁵ du = u⁶ / 6 + C

Resultado: (x³ + 4)⁶ / 6 + C

Reflexión

El método facilita la resolución de funciones compuestas cuando aparece una expresión junto con su derivada.

Tema 04

Integrales trigonométricas

Concepto

Estas integrales contienen funciones trigonométricas y pueden resolverse utilizando identidades fundamentales.

Identidades principales

sen²(x) + cos²(x) = 1
1 + tan²(x) = sec²(x)

Ejercicio desarrollado

Resolver: ∫ sen(x) cos³(x) dx

u = cos(x)

du = −sen(x) dx

−∫ u³ du

Resultado: −cos⁴(x) / 4 + C

Reflexión

Recordar las identidades trigonométricas permite transformar las expresiones antes de integrarlas.

Tema 05

Sustitución trigonométrica

Concepto

La sustitución trigonométrica se utiliza para transformar integrales con radicales que contienen sumas o diferencias de cuadrados.

Sustituciones principales

√(a² − x²): x = a sen(θ)
√(a² + x²): x = a tan(θ)
√(x² − a²): x = a sec(θ)

Ejemplo orientativo

Para una expresión como: √(16 − x²)

x = 4 sen(θ)

dx = 4 cos(θ) dθ

Se utiliza la identidad: 1 − sen²(θ) = cos²(θ)

Reflexión

Antes de seleccionar la sustitución es necesario identificar la forma exacta del radical.

Tema 06

Fracciones parciales

Concepto

Este método permite descomponer una función racional en fracciones simples que pueden integrarse individualmente.

Forma general

P(x) / Q(x) = A / (x − a) + B / (x − b)

Ejercicio desarrollado

Descomponer: 6 / [(x + 2)(x + 5)]

6 / [(x + 2)(x + 5)] = A / (x + 2) + B / (x + 5)

6 = A(x + 5) + B(x + 2)

A = 2

B = −2

Resultado: 2/(x + 2) − 2/(x + 5)

Reflexión

La factorización correcta del denominador es fundamental para determinar la forma de la descomposición.
